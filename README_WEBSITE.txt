WEBSITE VERSION (Dynamic Compilation)

This folder contains a Web Site style copy of the project (no .csproj needed).
Steps to use:
1. Copy Movies.mdb from ../App_Data into WebSiteVersion/App_Data.
2. Open Visual Studio -> File -> Open -> Web Site... -> select this WebSiteVersion folder.
3. Press F5 to run (Default.aspx is start page).

If you add code-behind files, ensure each .aspx directive uses CodeFile and the correct Inherits.
